# 📁 File Manager

A simple command-line **File Manager** built with **Python** that allows users to perform common file and directory operations through an interactive menu. This project demonstrates object-oriented programming (OOP), file system manipulation, and exception handling in Python.

---

## 🚀 Features

* 📂 List files and folders
* 📄 Create new files
* 🗑️ Delete files
* ✏️ Rename files
* 📋 Copy files
* 📦 Move files
* 📁 Create folders
* ❌ Delete folders
* 🔍 Search for files
* ℹ️ Display file information
* 🛡️ Error handling for invalid operations
* 🖥️ Easy-to-use command-line interface

---

## 🛠️ Technologies Used

* Python 3
* Built-in Python modules:

  * `os`
  * `shutil`
  * `pathlib`
  * `datetime`

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/mikiendialem/file-manager.git
```

### 2. Navigate to the project folder

```bash
cd file-manager
```

### 3. Run the application

```bash
python main.py
```

---

## 📖 How to Use

After starting the application, you will see a menu similar to the following:

```text
==============================
      FILE MANAGER
==============================

1. List Files
2. Create File
3. Delete File
4. Rename File
5. Copy File
6. Move File
7. Create Folder
8. Delete Folder
9. Search File
10. File Information
11. Exit

Choose an option:
```

Select a menu option and follow the prompts to perform file management tasks.
